# Creative Generator — Copies e Criativos para Instagram

## Quando Usar

- Quando o usuario precisar de copies para anuncios
- Quando o lead-gen-creator precisar de conteudo criativo
- Quando o campaign-analyzer recomendar testar novos criativos
- Quando o usuario pedir variações de anuncios

## Contexto do Produto

**Produto:** Claude Code via API
**Promessa:** Pague R$200 ao inves de US$200 — use mais, pague menos
**Beneficios:**
- Acesso ao Claude Opus (modelo mais potente)
- Economia de ate 80% vs plano Max
- Uso flexivel via API — pague pelo que usar
- Ferramenta de produtividade para desenvolvedores
- Assistente de codigo inteligente direto no terminal

**Publico:** Desenvolvedores e profissionais de tech no Brasil

## Processo

### Passo 1: Definir Angulo do Criativo

Apresentar os angulos disponiveis:

#### Angulo 1: Economia
Foco na diferenca de preco — R$200 vs US$200
```
Hook: "Voce esta pagando caro demais pelo seu assistente de IA"
Problema: Plano Max custa US$200/mes (~R$1.000)
Solucao: Claude Code via API por R$200/mes
Prova: Mesmo modelo (Opus), mesma qualidade
```

#### Angulo 2: Produtividade
Foco em como o Claude Code acelera o trabalho
```
Hook: "Desenvolvedores usando Claude Code entregam 3x mais rapido"
Problema: Tarefas repetitivas consomem horas
Solucao: Claude Code faz o trabalho pesado por voce
Prova: Refatoracao, debug, testes — tudo no terminal
```

#### Angulo 3: Exclusividade/Hack
Foco no "segredo" que poucos conhecem
```
Hook: "O hack que devs brasileiros estao usando para pagar 5x menos no Claude"
Problema: Precos em dolar sao proibitivos
Solucao: Usar via API e pagar em reais
Prova: Funcionalidade completa por uma fracao do preco
```

#### Angulo 4: FOMO / Tendencia
Foco em nao ficar para tras
```
Hook: "Enquanto voce ainda coda manual, seus concorrentes usam IA"
Problema: O mercado esta mudando rapido
Solucao: Claude Code e o copiloto que voce precisa
Prova: Milhares de devs ja migraram
```

### Passo 2: Gerar Copies

Para cada angulo selecionado, gerar 3 variacoes:

#### Formato Instagram Feed (125 caracteres principal + 2000 corpo)

```
VARIACAO A (Curta - Impacto Direto):
{headline de ate 40 caracteres}

{copy de ate 125 caracteres focada no hook}

{CTA claro}

VARIACAO B (Media - Problema/Solucao):
{headline}

{identificacao do problema - 1 linha}
{agitacao - 1 linha}
{solucao - 1 linha}
{beneficio - 1 linha}

{CTA}

VARIACAO C (Longa - Storytelling):
{headline}

{historia ou contexto - 2-3 linhas}
{virada/descoberta - 1-2 linhas}
{beneficios em lista - 3-4 itens}
{prova social ou urgencia - 1 linha}

{CTA}
```

#### Formato Instagram Stories (texto curto + visual)

```
Slide 1 (Hook): {pergunta ou afirmacao impactante - max 60 chars}
Slide 2 (Problema): {dor do publico - max 60 chars}
Slide 3 (Solucao): {como o produto resolve - max 60 chars}
Slide 4 (CTA): {chamada para acao - max 40 chars} + Swipe Up
```

#### Formato Instagram Reels (roteiro)

```
0-3s: Hook visual + texto "{frase de impacto}"
3-8s: Problema "{contexto da dor}"
8-15s: Solucao "{demonstracao do produto}"
15-25s: Beneficios "{3 pontos rapidos}"
25-30s: CTA "{chamada para acao + formulario}"
```

### Passo 3: Headlines

Gerar 10 opcoes de headlines seguindo formulas comprovadas:

1. **Numero + Beneficio:** "5 motivos para trocar o Max pelo API"
2. **Como + Resultado:** "Como pagar 5x menos pelo Claude Opus"
3. **Pergunta:** "Cansado de pagar US$200/mes?"
4. **Segredo/Hack:** "O segredo que devs usam para economizar no Claude"
5. **Comparacao:** "R$200 vs US$200 — Mesmo Claude, preco diferente"
6. **Urgencia:** "Antes que o preco suba: Claude Code por R$200"
7. **Social Proof:** "Milhares de devs BR ja migraram — e voce?"
8. **Desafio:** "Teste o Claude Code por 7 dias e comprove"
9. **Negacao:** "Nao pague US$200 se voce pode pagar R$200"
10. **Resultado:** "De junior a senior: como o Claude Code transforma seu codigo"

### Passo 4: Sugestoes Visuais

Para cada criativo, sugerir:

- **Formato de imagem:** 1080x1080 (Feed), 1080x1920 (Stories/Reels)
- **Estilo visual:** Dark mode (tech), terminal/codigo, split screen comparativo
- **Elementos obrigatorios:** Logo, headline legivel, CTA visivel
- **Cores sugeridas:** Fundo escuro (#1a1a2e), Accent laranja (#ff6b35) ou azul (#4a9eff)
- **Tipografia:** Sans-serif moderna, bold para headline

### Passo 5: Testes A/B

Recomendar estrutura de teste:

```
Teste 1: Angulos diferentes (mesmo formato)
- Ad A: Angulo Economia
- Ad B: Angulo Produtividade
- Ad C: Angulo Hack

Teste 2: Formato de copy (mesmo angulo)
- Ad A: Copy curta
- Ad B: Copy media
- Ad C: Copy longa

Teste 3: Headlines (mesmo angulo e formato)
- Ad A: Headline pergunta
- Ad B: Headline numero
- Ad C: Headline comparacao
```

## Frameworks de Copywriting

### PAS (Problem - Agitation - Solution)
1. Identifique a dor
2. Amplifique a dor
3. Apresente a solucao

### AIDA (Attention - Interest - Desire - Action)
1. Capture atencao com hook forte
2. Gere interesse com dados/fatos
3. Crie desejo com beneficios
4. Direcione para acao

### BAB (Before - After - Bridge)
1. Situacao antes (dor)
2. Situacao depois (transformacao)
3. Ponte (como chegar la = produto)

## Restricoes

- Copies devem respeitar as Politicas de Publicidade do Meta
- NAO fazer promessas exageradas ou irreais
- NAO usar linguagem enganosa sobre precos ou funcionalidades
- NAO fazer comparacoes diretas difamatorias com concorrentes
- Respeitar limites de caracteres de cada formato
- Sempre incluir disclaimer quando necessario (ex: "precos podem variar")
- NAO usar clickbait puro — o conteudo deve entregar o que promete
